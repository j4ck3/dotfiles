# dotfiles

Personal configs for **CachyOS** (Arch-based).

**Desktop:** Hyprland, Waybar, UWSM, Walker + Elephant, Helium Browser, AGS (volume/media/power popups). Optional **hypr_htpc** profile for HTPC.

**Shell & prompt:** Bash, Starship.

**Editors & coding:** Lazyvim, Cursor, OpenCode, t3code, Ghostty.

**CLI:** tmux, lazygit, lazydocker, btop, just.

**Apps:** Spotify & Discord (Flatpak), Betterbird, FreeCAD.

**Network / infra:** Docker, Tailscale, Netbird.

**Dotfiles layout:** GNU Stow package dirs (`bash/`, `hypr/`, `nvim/`, …). Package lists: `packages.txt`, `aur-packages.txt`, `flatpak-packages.txt`.
